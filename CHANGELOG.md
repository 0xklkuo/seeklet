# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-31

### Added
- initial Seeklet MVP
- seeded crawler with host scoping
- `robots.txt` support
- HTML extraction for title, text, and links
- URL normalization
- SQLite-backed local index
- BM25 search ranking
- snippet generation
- CLI commands for crawl, search, stats, and reset
- tests, linting, and CI
- contributor and architecture documentation
